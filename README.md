A simple Haskell fighting game project.

## Commands

- **Build:** `make build`  
  Compiles the source files into the `build/` directory.

- **Run:** `make run`  
  Builds (if needed) and runs the game executable.

- **Clean:** `make clean`  
  Removes all compiled files and the `build/` directory.
